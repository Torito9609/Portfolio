Proyectos

Proyecto 1: E-commerce El Zafiro Azul

Una plataforma de comercio electrónico completamente funcional donde los usuarios pueden navegar, agregar productos al carrito y realizar compras. Enfocada en la optimización del backend para una búsqueda rápida de productos y un proceso de pago seguro.

Tecnologías usadas:
  • Java
  • Spring Boot
  • JPA
  • Hibernate
  • JWT
  • MySQL
  • HTML
  • CSS
  • JavaScript
Ver repo Front End Ver repo Back End
Detalles adicionales:

El sitio web de comercio electrónico utiliza Spring Boot para el backend, lo que permite un desarrollo rápido e inyección de dependencias. Usé Hibernate y JPA para el ORM, lo que permite una interacción fluida con la base de datos MySQL. Se utiliza JWT para una autenticación segura de usuarios. El front-end está construido con HTML, CSS y JavaScript, asegurando una interfaz receptiva y fácil de usar.
